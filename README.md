# Computational Finance Lab

A modular Python project for computational finance and stochastic modelling.

The goal of this project is to refactor verified computational finance implementations into a reusable Python package.

Planned components include:

- Binomial / CRR option pricing
- Black-Scholes pricing
- Numerical integration methods
- Monte Carlo simulation
- Random number generation methods
- Barrier options
- American options
- Fourier and Laplace transform pricing
- Implied volatility and calibration
- Stochastic process simulation
